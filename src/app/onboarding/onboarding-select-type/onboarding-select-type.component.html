(Gnosis) Safe

Enable your Safe to send / receive assets (ERC20 Tokens, NFTs, ...) & execute transactions on remote blockchains.

Useful for treasuries wanting to manage assets on many chains, while updating the multisig structure on only one.

Externally Owned Account

Select this if you're using a non-smart contract wallet such as MetaMask, Rabby, Coinbase Wallet, etc...

Useful if you want to execute transactions on multiple chains, but pay for gas on only one.

Other smart contract

Klaster cross-chain accounts work with any MutliSig, Smart Contract Wallet & even with generic EVM smart contracts.
